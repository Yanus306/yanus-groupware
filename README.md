# yANUs - 동아리 그룹웨어

React PWA 기반 동아리 그룹웨어입니다. 웹과 모바일에서 모두 사용할 수 있습니다.

## 기능

- **업무 채팅**: 채널 및 다이렉트 메시지
- **출퇴근**: 출근/퇴근 기록 및 개인 스케줄 설정
- **Set Work Days (본인만)**: 출/퇴근 시간, 근무 요일을 본인 계정으로만 수정 가능
- **관리자 기능**: 팀장/리더는 출퇴근 기록 열람
- **파일 공유 드라이브**: 폴더/파일 공유
- **AI 챗봇**: 파일 드라이브 기반 질의응답
- **Sticky 채팅/챗봇**: 웹 화면 어디서든 우측 하단에서 접근 가능

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
npm run preview  # 빌드 결과 미리보기
```

## PWA

- `npm run build` 후 `dist` 폴더를 정적 호스팅하면 PWA로 설치 가능
- 모바일에서 홈 화면에 추가 시 앱처럼 실행
